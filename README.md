# Симулятор Скама

Проект создан для публикации на платформе **Yandex Games** для коммерции.

**Платформа:** WebGL (для ПК и мобильных устройств)

## 🎮 О проекте

**Симулятор Скама** — это увлекательная игра-кликер, в которой вы становитесь мастером обмана. Ваша задача — фармить валюту, придумывать хитрые схемы и "разводить" персонажей: от доверчивых бабулек до подозрительных хакеров. Каждый новый босс — умнее предыдущего, и вам предстоит проявить всю свою хитрость, чтобы победить.

**Важно:** Это всего лишь игра. Все события вымышлены.

## 🎯 Основные механики

- **Кликер:** Нажимайте на экран, чтобы атаковать и зарабатывать монеты.
- **Улучшения:** Тратьте монеты на улучшение урона за клик.
- **Боссы:** Побеждайте боссов с уникальными характерами за ограниченное время (40 секунд).
- **Бонусы:** Смотрите рекламу, чтобы получать бонусные монеты.

## 🚀 Сборка и запуск

1. Скачайте репозиторий.
2. Откройте проект в Unity (версию смотрите в `ScamSimulator/ProjectSettings/ProjectVersion.txt`).
3. Для запуска в редакторе откройте сцену `ScamSimulator/Assets/Scenes/MainScene.unity` и нажмите `Play`.
4. Для публикации на Yandex Games соберите проект в **WebGL** через `File → Build Settings`.
5. Загрузите сборку в личном кабинете разработчика Yandex Games.

# Scam Simulator

The project was created for the **Yandex Games** platform in profit.

**Platform:** WebGL (for PC and mobile devices)

## 🎮 About the project

**Scam Simulator** is an engaging clicker game where you become a master of deception. Your goal is to farm currency, come up with cunning schemes, and "scam" various characters, from trusting elderly people to suspicious hackers. Each new boss is smarter than the last, and you'll need to use all your cunning to win.

**Important:** This is just a game. All events are fictional.

## 🎯 Core mechanics

- **Clicker:** Click the screen to attack and earn coins.
- **Upgrades:** Spend coins to increase your damage per click.
- **Bosses:** Defeat unique bosses within a time limit (40 seconds).
- **Bonuses:** Watch ads to earn bonus coins.
